# Welcome to Specktral!

"Alles ist besser mit Speck" (Robert Speck), translates to "Everything is better with bacon" and we don't dare argue with that.

Specktral is very much under development, but it will eventually be a lightweight code for generating spectral discretizations, analogously to [qmat](https://github.com/Parallel-in-Time/qmat) for quadrature and SDC preconditioner coefficients.
The intent of Specktral to enable seamless use of spectral methods in any code by providing discretization matrices via an easy to use interface.
Specktral is not a competitor to [Dedalus](https://github.com/DedalusProject/dedalus), which is a powerful spectral-method simulation framework, but which doesn't readily expose the discretization matrices for further use.
